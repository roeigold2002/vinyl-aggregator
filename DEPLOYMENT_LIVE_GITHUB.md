## 🆘 Troubleshooting

**"Railway deploy fails"**
- Check Railway token is correct
- Verify backend/requirements.txt exists
- Check Railway logs on dashboard

**"Vercel deploy fails"**
- Check Vercel token is correct
- Verify frontend/package.json exists
- Check Vercel logs on dashboard

**"Frontend can't connect to backend"**
- Set VITE_API_URL in frontend environment on Vercel
- Example: `VITE_API_URL=https://your-railway.railway.app`

**"Database connection error"**
- Railway creates DATABASE_URL automatically
- Set it in Railway environment variables
- Check PostgreSQL is running
